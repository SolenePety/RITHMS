# Taxonomy assignation

Taxonomy assignation

## Usage

``` r
taxonomy
```

## Format

A `data.frame` containing the taxonomy for each OTU sequenced matching
the ID in the microbiota matrix till Kingdom level.

## Source

[Déru et al.](https://pmc.ncbi.nlm.nih.gov/articles/PMC9194801/) for
microbiota data and corresponding metadata
